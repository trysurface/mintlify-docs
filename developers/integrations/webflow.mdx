### 1. Create a new Surface Form

Create a new form using our no code [form builder](https://app.withsurface.com) and copy the form sharing link. You can also follow [this](../quick-start/create-first-form) tutorial to build a basic Surface Form. <br/> For this tutorial, suppose it is: https://forms.withsurface.com/s/cm2jrdp7o0001jv03ui09zrgy

### 2. Embedding Form in Webflow

There are two ways to embed the Surface Form, you may choose one according to your preference.

<Tabs>
    <Tab title="Inline Embed">
        To embed the Surface form inline, go to the Webflow design studio and select the section where the form will be added.
        After selecting the section, you should see the `Style` tab in the right panel.
        <img className="block" src="/images/webflow/webflow-design-studio.png" alt="webflow" />
        Now add `surface-form-target` in the Style Selector:
        <img className="block" src="/images/webflow/webflow-design-studio-inline.png" alt="webflow" />
    </Tab>
    <Tab title="Popover">
        To configure the Popover, navigate to your Webflow Design Studio and select the button that triggers the form.
        After Selecting the button, you should see `Style` tab on your right panel.
        <img className="block" src="/images/webflow/webflow-design-studio.png" alt="webflow" />
        Now add `surface-form-button` in Style Selector:
        <img className="block" src="/images/webflow/webflow-design-studio-popslide.png" alt="webflow" />
    </Tab>
    <Tab title="Slideover">
        To configure the Slideover, navigate to your Webflow Design Studio and select the button that triggers the form.
        After Selecting the button, you should see `Style` tab on your right panel.
        <img className="block" src="/images/webflow/webflow-design-studio.png" alt="webflow" />
        Now add `surface-form-button` in Style Selector:
        <img className="block" src="/images/webflow/webflow-design-studio-popslide.png" alt="webflow" />
    </Tab>

</Tabs>

### 3. Adding JavaScript code

Go to the webflow [dashboard](https://webflow.com/dashboard) > `Project Section` > Select `Custom Code` and add the following custom code into the `head` code section.

<Tabs>
    <Tab title="Inline Embed">
        ```html
        <script>
            window.onload = function() {
                const surface_src = "<form url here>" // replace this with your Surface Form URL.
                const surface_embed_type = "inline"
                const target_element_class = "surface-form-target"
                const c = new SurfaceEmbed(surface_src, surface_embed_type, target_element_class)

                c.embedSurfaceForm();
            }
        </script>
        ```
    </Tab>
    <Tab title="Popover">
            ```html
            <script>
                window.onload = function() {
                    const surface_src = "<form url here>" // replace this with your Surface Form URL.
                    const surface_embed_type = "popup"
                    const target_element_class = "surface-form-button"
                    const c = new SurfaceEmbed(surface_src, surface_embed_type, target_element_class)

                    c.embedSurfaceForm();
                }
            </script>
            ```
            You can check this [guide](../surface-forms/popup-slideover) for a more detailed explanation about Popover Embedding.
    </Tab>
    <Tab title="Slideover">
        ```html
        <script>
            window.onload = function() {
                const surface_src = "<form url here>" // replace this with your Surface Form URL.
                const surface_embed_type = "slideover"
                const target_element_class = "surface-form-button"
                const c = new SurfaceEmbed(surface_src, surface_embed_type, target_element_class)

                c.embedSurfaceForm();
            }
        </script>
        ```
        You can check this [guide](../surface-forms/popup-slideover) for a more detailed explanation about Slideover Embedding.
    </Tab>
</Tabs>

### 4. Submit all Changes

Your Surface Form should now be embedded on your Webflow website!
