### 1. Create a new Surface Form

Create a new form using our no code [form builder](https://app.withsurface.com) and copy the form sharing link. You can also [follow this](../quick-start/create-first-form) tutorial to create a basic Surface Form. <br/> For this tutorial suppose its https://forms.withsurface.com/s/cm2jrdp7o0001jv03ui09zrgy

### 2. Embedding Form in Webflow.

There are 2 ways to embed the Surface Form, you may choose one according to your preference.

<Tabs>
    <Tab title="Inline Embed">
        For Inline Embedding, go to your webflow design studio and select section where you need your form to be.
        After Selecting the section, you should see `Style` tab on your right panel.
        <img className="block" src="/images/webflow/webflow-design-studio.png" alt="webflow" />
        Now add `surface-form-target` in Style Selector, That should look something like this:
        <img className="block" src="/images/webflow/webflow-design-studio-inline.png" alt="webflow" />
    </Tab>
    <Tab title="Popover/Slideover">
        To configure the Popover/Slideover, navigate to your Webflow Design Studio and select the button that triggers the form.
        After Selecting the button, you should see `Style` tab on your right panel.
        <img className="block" src="/images/webflow/webflow-design-studio.png" alt="webflow" />
        Now add `surface-form-button` in Style Selector, That should look something like this:
        <img className="block" src="/images/webflow/webflow-design-studio-popslide.png" alt="webflow" />
    </Tab>

</Tabs>

### 3. Adding JavaScript code

Go to webflow [dashboard](https://webflow.com/dashboard) > Go to `Project Section` > Select `Custom Code` and add the following custom code into the Head code section.

<Tabs>
    <Tab title="Inline Embed">
        ```javascript
        <script>
            window.onload = function() {
                const surface_src = "<form url here>" // replace this with your Surface Form URL.
                const surface_embed_type = "inline"
                const target_element_class = "surface-form-target"
                const c = new SurfaceEmbed(surface_src, surface_embed_type, target_element_class)

                c.embedSurfaceForm();
            }
        </script>
        ```
    </Tab>
    <Tab title="Popover/Slideover">
        <Accordion title="Popover">
            ```javascript
            <script>
                window.onload = function() {
                    const surface_src = "<form url here>" // replace this with your Surface Form URL.
                    const surface_embed_type = "popup"
                    const target_element_class = "surface-form-button"
                    const c = new SurfaceEmbed(surface_src, surface_embed_type, target_element_class)

                    c.embedSurfaceForm();
                }
            </script>
            ```
            You can check this [guide](../surface-forms/popup-slideover) for detailed explaination about Popover Embedding.
        </Accordion>
        <Accordion title="Slideover">

            ```javascript
            <script>
                window.onload = function() {
                    const surface_src = "<form url here>" // replace this with your Surface Form URL.
                    const surface_embed_type = "slideover"
                    const target_element_class = "surface-form-button"
                    const c = new SurfaceEmbed(surface_src, surface_embed_type, target_element_class)

                    c.embedSurfaceForm();
                }
            </script>

            You can check this [guide](../surface-forms/popup-slideover) for detailed explaination about Slideover Embedding.

            ```
        </Accordion>

    </Tab>
</Tabs>
