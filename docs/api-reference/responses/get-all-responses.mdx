---
title: "Get All Responses"
description: "Retrieve all form responses for your environment"
---

<Endpoint method="GET" path="/api/v1/responses" summary="Get all responses for an environment">
  <ParamField header="x-api-key" type="string" required>
    Your API key for authentication.
  </ParamField>
</Endpoint>


Returns all form responses across all forms in your environment. By default, only completed responses are returned.


## Request
<Tip> Learn how to create an API key in the [API Keys](/docs/api-reference/api-keys) documentation. </Tip>

<CodeGroup>
```bash cURL
curl -X GET 'https://forms.withsurface.com/api/v1/responses' \
  -H 'x-api-key: your-api-key-here'
```

```typescript TypeScript
const response = await fetch('https://forms.withsurface.com/api/v1/responses', {
  method: 'GET',
  headers: {
    'x-api-key': 'your-api-key-here',
  },
});

if (!response.ok) {
  throw new Error(`HTTP error! status: ${response.status}`);
}

const data = await response.json();
```
</CodeGroup>

## Response

<ResponseField name="responses" type="array">
  Array of response objects for all forms in your environment.
  
  <Expandable title="Response object properties">
    <ResponseField name="id" type="string">
      Unique identifier for the response.
    </ResponseField>
    
    <ResponseField name="formId" type="string">
      ID of the form this response belongs to.
    </ResponseField>
    
    <ResponseField name="submittedAt" type="string">
      ISO 8601 timestamp of when the response was submitted.
    </ResponseField>
    
    <ResponseField name="data" type="object">
      Object containing the form field values submitted by the respondent.
    </ResponseField>
  </Expandable>
</ResponseField>
