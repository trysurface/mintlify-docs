Surface Forms integrates seamlessly with HubSpot to automatically create and update contacts, companies, and deals from your form submissions.

## Integration Steps

<Frame>
![HubSpot Integration Setup](/images/integrations/hubspot/1.png)
</Frame>

1. Navigate to the **Workflow Builder**
2. Add any HubSpot action and click **Connect**
3. Select your HubSpot organization when prompted
4. Accept Surface Labs invite
5. Click **Connect App** to complete setup

## Setting Up Workflows

1. Go to **Workflow Builder** from the sidebar
2. Click "Create workflow"
3. Select source:
    - Choose "Surface forms" as source
    - Select your form
    - Choose trigger type (Finished/In Progress/Dropped Off)
4. Add HubSpot action:
    - Click "+ Actions"
    - Select "HubSpot" from the actions list
        
        <Frame>
        ![HubSpot Actions Selection](/images/integrations/hubspot/2.png)
        </Frame>

## Available Actions

1. **Create Contact:** Create a new contact in HubSpot from form submission data.
2. **Create or Update Contact:** Create or update a contact in HubSpot using form submission data.
3. **Create Company:** Create a new company in HubSpot.
4. **Create Form Response:** Create a form response in HubSpot.
5. **Create Deal:** Create a new deal in HubSpot and associate it with contacts or companies.
6. **Get Owner:** Retrieve a HubSpot owner by email address.
7. **HubSpot AI Agent:** Use the AI Agent to manage contacts and deals in HubSpot.
