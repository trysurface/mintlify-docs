## 1. Login to Surface Dashboard
Navigate to [Surface Dashboard](https://app.withsurface.com/auth/login) and login with your credentials. If you don't have an account, you can [sign up here](https://app.withsurface.com/auth/signup).

## 2. Create a New Form
<Frame>
  <img className="block" src="/images/first-form/1.png" alt="first form" />
</Frame>
Go to `Forms` tab and Click on the `Create New Form` button on the navbar to create a new form.

### Start from scratch or choose a template
<Frame>
  <img className="block" src="/images/first-form/image.png" alt="first form" />
</Frame>

## 3. Select a Layout
<Frame>
  <img className="block" src="/images/first-form/2.png" alt="first form" />
</Frame>
Choose a layout for your form from the following options:
- **Single Pane**: A simple form layout with a single column.
- **Dual Pane**: A form layout with two columns, great for marketing.

## 4. Build the Form

### Overview
<Frame>
  <img className="block" src="/images/first-form/3.png" alt="first form" />
</Frame>
Surface Forms provides an initial template to start from, you can start editing the same or create new one!

### Form Steps
<Frame>
    <img className="block" src="/images/first-form/4.png" alt="first form" />
</Frame>
Form Steps are listed on the left side of the form builder under `Steps` section.

#### Add new steps
To add a new step, click on the `+ Add step` button or copy an existing step by clicking on the `Copy step` button.

### Edit the form step
<Frame>
    <img className="block" src="/images/first-form/5.png" alt="first form" />
</Frame>

#### Add new components
To add a new component, click on the `+ Add component` button or copy an existing component from the form by clicking on the `Copy existing component` button.

#### Customize Layout
<Frame>
    <img className="block" src="/images/first-form/6.png" alt="first form" />
</Frame>
Both `Questions` and `Marketing` panel have layout customization options.
Some of the customization options are:
- `Horizontal Padding`
- `Max Width of the pane`

#### Marketing Pane layout customization
<Frame>
    <img className="block" src="/images/first-form/7.png" alt="first form" />
</Frame>
Marketing pane extends some more customization options like:
1. Customizing `Marketing Pane` width
2. Adding `Background Image`
3. Adding `Logo` in header:
    - Expand `Add Component` dropdown and select `Image` component to add logo in marketing header.

### Question Pane and Marketing Pane
Each Step is divided into 2 tabs `Questions` and `Marketing`.
    - `Questions` tab is where you can add multiple components to the form like `Input Fields`.
    - `Marketing` tab is where you can add multiple marketing components to form.

### Edit a component
<Frame>
    <img className="block" src="/images/first-form/8.png" alt="first form" />
</Frame>
To edit a component, either click on the component directly from the form preview or click on the component from the components list under right side-panel.

### Style a component
<Frame>
    <img className="block" src="/images/first-form/9.png" alt="first form" />
</Frame>
Each component has a `Style` panel to style the component. Each component has different style options.

Some of the style options are:
- `Background Color`
- `Text Color`
- `Border Color`
- `Border Width`
- `Border Radius`
- `Box Shadow`

## 5. Save and Share 
    Change the form name on the `navbar` and Save the form by clicking on the `Save` button.
    Share the form using `Share` button on the navbar.
    - You can share the form using the `Share Link` or `Embed Code` to embed the form on your website.
        <Frame>
            <img className="block" src="/images/first-form/10.png" alt="first form" />
        </Frame>
    - Read more about [embedding Surface Form](/developers) on your website.
