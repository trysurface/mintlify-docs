## Create a New Form and Router in RevenueHero

### Create a New Form
<Frame>
    <img src="/images/integrations/revenue-hero/1.png" />
</Frame>
- Navigate to `Inbound` > `Forms` tab and create a new RevenueHero form.
- Ensure that you add all the required fields for your form.

### Create a New Router
<Frame>
    <img src="/images/integrations/revenue-hero/2.png" />
</Frame>
- Navigate to `Inbound` > `Routes` tab and create a new RevenueHero router.

### Retrieve the Router ID
Once you have created the router, you can obtain the router ID in the following ways:

1. Navigate to the `Routers` tab and copy the ID from the URL.
2. Navigate to the `Routers` tab, click on `Widget Installation` in the navigation menu, and retrieve the Router ID from the code snippet.
<Tip>
    The Router ID is the value of the `routerId` variable in the code snippet.
</Tip>

## Configure the Scheduler Component
<Frame>
    <img src="/images/integrations/revenue-hero/3.png" />
</Frame>

1. Enter the Router ID from RevenueHero into the `RevenueHero Router ID` field.
2. Map the form fields accordingly.
