Complete these steps to start converting more website traffic into revenue with Surface Labs.

## Step 1: Install the Surface Tag

Connect Surface Forms to your website to start capturing visitor data. There are several supported methods of integration:

<Note>
  Check out the [How to Get Site ID](./surface-tag/how-to-get-site-id) page to get your site ID.
</Note>

<Card title="Surface Tag" icon="code">
  Add the Surface Forms JavaScript snippet to your website with minimal setup:
  ```javascript
  <!-- Start Surface Tag -->
  <script 
    src="https://cdn.jsdelivr.net/gh/trysurface/scripts@latest/surface_tag.min.js"
    siteId="YOUR_SITE_ID"> // Replace with your site ID
  </script>
  <!-- End Surface Tag -->
  ```
</Card>
