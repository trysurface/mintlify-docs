---
title: How to Get Site ID
description: Learn how to get your site ID for the Surface Tag.
---

<Steps>
  <Step title="Go to the Settings page">
    <Frame>
        <img src="/images/settings.png" alt="Settings" />
    </Frame>
  </Step>
  <Step title="Get your site ID under Overview Tab">
    <Frame>
        <img src="/images/site-id.png" alt="Site ID" />
    </Frame>
  </Step>
  <Step title="Add the site ID to the Surface Tag">
    ```javascript
    <!-- Start Surface Tag -->
    <script 
        src="https://cdn.jsdelivr.net/gh/trysurface/scripts@latest/surface_tag.min.js"
        siteId="YOUR_SITE_ID"> // Replace with your site ID
    </script>
    <!-- End Surface Tag -->
    ```
  </Step>
</Steps>
