<Note>

Add the Surface Tag to your website by including this script in your HTML:

```javascript
<!-- Start Surface Tag -->
<script 
    src="https://cdn.jsdelivr.net/gh/trysurface/scripts@latest/surface_tag.min.js" 
    siteId="YOUR_SITE_ID">
</script>
<!-- End Surface Tag -->
```

## Install with Google Tag Manager

To install the Surface Tag using Google Tag Manager (GTM):

1. In your GTM workspace, click **Add a new tag**.
2. Choose **Tag Type**: select **Custom HTML**.
3. Paste the following code into the HTML field (replace `YOUR_SITE_ID` with your actual Site ID):

```html
<!-- Start Surface Tag -->
<script>
(function () {
  var s = document.createElement('script');
  s.type = 'application/javascript';
  s.src = 'https://cdn.jsdelivr.net/gh/trysurface/scripts@latest/surface_tag.min.js';
  s.setAttribute('siteId', 'YOUR_SITE_ID');
  document.head.appendChild(s);
})();
</script>
<!-- End Surface Tag -->
```

4. Set the trigger to **Page View** > **Initialization** (so the tag loads as soon as possible).
5. Save and publish your changes in GTM.

<Note>
  You can find your Site ID on the [How to Get Site ID](./how-to-get-site-id) page.
</Note>
