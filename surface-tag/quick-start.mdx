### Add Surface Tag

Add the below script tag inside your `<head>` tag.

```html
<!-- Start Surface Tag -->
<script src="https://cdn.jsdelivr.net/gh/trysurface/scripts@latest/surface_tag.min.js"></script>
<!-- End Surface Tag -->
```
